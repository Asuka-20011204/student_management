# 学生管理系统 - 需求分析文档

## 1. 项目概述

### 1.1 项目背景
面向学校的 Web 端学生信息管理系统，实现学生、班级、课程、成绩的数字化管理。

### 1.2 技术选型

| 层级 | 技术 |
|------|------|
| 后端框架 | Spring Boot 2.7 |
| 前端框架 | Vue 3 + Element Plus |
| 数据库 | MySQL 8.0 |
| ORM | MyBatis-Plus |
| 接口规范 | RESTful API |
| 认证方案 | Spring Security + JWT |

### 1.3 用户角色

| 角色 | 权限概述 |
|------|----------|
| 管理员 | 系统全部功能，管理用户、角色分配 |
| 教师 | 查看所教班级学生信息，录入与修改成绩 |
| 学生 | 查看个人基本信息和各科成绩 |

---

## 2. 功能模块

### 2.1 登录与用户管理
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 登录/登出 | 账号密码登录，JWT 认证，Token 过期自动跳转登录页 | P0 |
| 用户管理 | 管理员可创建、编辑、禁用系统用户，并分配角色 | P0 |
| 修改密码 | 登录用户修改自己的密码 | P1 |

### 2.2 学生管理
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 学生列表 | 分页查询，支持按姓名/学号/班级条件筛选 | P0 |
| 新增/编辑/删除 | 学生基本信息的 CRUD | P0 |
| 批量删除 | 勾选多条记录后批量删除 | P1 |

### 2.3 班级管理
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 班级列表 | 分页查询，显示班级人数 | P0 |
| 新增/编辑/删除 | 班级信息的 CRUD | P0 |
| 班级学生列表 | 点击班级查看该班所有学生 | P1 |

### 2.4 课程管理
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 课程列表 | 分页查询所有课程 | P1 |
| 新增/编辑/删除 | 课程信息（名称、编号、学分、课时）的 CRUD | P1 |
| 排课 | 为班级分配课程并指定授课教师 | P1 |

### 2.5 成绩管理
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 成绩录入 | 教师按「班级 + 课程」维度批量录入学生成绩 | P0 |
| 成绩查询 | 按学生/班级/课程多维度查询 | P0 |
| 成绩统计 | 按班级+课程计算平均分、最高分、最低分、及格率 | P1 |
| 学生成绩单 | 学生端查看自己各科成绩 | P1 |

---

## 3. 非功能性要求

| 指标 | 要求 |
|------|------|
| 响应时间 | 普通查询 < 500ms |
| 密码安全 | BCrypt 加密存储 |
| 接口鉴权 | 除登录外所有接口需携带 JWT Token |
| 统一响应 | 所有接口返回 `{ code, message, data }` 格式 |
| 分页格式 | `{ records, total, page, pageSize }` |

---

## 4. 界面规划

- **整体布局**：侧边栏菜单 + 顶栏（用户信息/退出）+ 主体内容区
- **登录页**：居中卡片表单
- **列表页**：搜索区 + 操作按钮区 + 数据表格 + 底部分页
- **表单**：弹窗（Dialog）方式编辑，简单高效
- **成绩统计**：ECharts 柱状图/饼图展示
