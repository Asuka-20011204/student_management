## 2. 表结构（共8张表）

### 2.1 sys_user — 系统用户表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| username | VARCHAR(50) | NOT NULL, UNIQUE | 用户名 |
| password | VARCHAR(200) | NOT NULL | 密码（BCrypt） |
| real_name | VARCHAR(50) | NOT NULL | 真实姓名 |
| phone | VARCHAR(20) | — | 手机号 |
| email | VARCHAR(100) | — | 邮箱 |
| status | TINYINT | NOT NULL, DEFAULT 1 | 状态：0=禁用, 1=启用 |
| create_time | DATETIME | NOT NULL | 创建时间 |
| update_time | DATETIME | — | 更新时间 |

### 2.2 sys_role — 角色表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| role_name | VARCHAR(50) | NOT NULL | 角色名称 |
| role_code | VARCHAR(50) | NOT NULL, UNIQUE | 角色编码（ADMIN/TEACHER/STUDENT） |
| description | VARCHAR(255) | — | 描述 |
| create_time | DATETIME | NOT NULL | 创建时间 |

### 2.3 sys_user_role — 用户角色关联表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| user_id | BIGINT | NOT NULL, FK → sys_user.id | 用户ID |
| role_id | BIGINT | NOT NULL, FK → sys_role.id | 角色ID |

> 设计说明：一个用户可拥有多个角色（如既是教师又是管理员），前端根据角色集合控制菜单显示。

### 2.4 class — 班级表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| class_name | VARCHAR(100) | NOT NULL | 班级名称 |
| grade | VARCHAR(20) | NOT NULL | 年级 |
| head_teacher_id | BIGINT | FK → sys_user.id | 班主任ID |
| room | VARCHAR(50) | — | 教室号 |
| description | VARCHAR(500) | — | 班级描述 |
| create_time | DATETIME | NOT NULL | 创建时间 |
| update_time | DATETIME | — | 更新时间 |

### 2.5 student — 学生表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| student_no | VARCHAR(30) | NOT NULL, UNIQUE | 学号 |
| name | VARCHAR(50) | NOT NULL | 姓名 |
| gender | TINYINT | DEFAULT 0 | 性别：0=未知, 1=男, 2=女 |
| birthday | DATE | — | 出生日期 |
| phone | VARCHAR(20) | — | 联系电话 |
| email | VARCHAR(100) | — | 邮箱 |
| address | VARCHAR(255) | — | 家庭住址 |
| class_id | BIGINT | FK → class.id | 所属班级 |
| enrollment_date | DATE | — | 入学日期 |
| status | TINYINT | DEFAULT 1 | 0=休学, 1=在读, 2=毕业 |
| user_id | BIGINT | FK → sys_user.id | 关联登录账号（可选） |
| create_time | DATETIME | NOT NULL | 创建时间 |
| update_time | DATETIME | — | 更新时间 |

### 2.6 course — 课程表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| course_name | VARCHAR(100) | NOT NULL | 课程名称 |
| course_code | VARCHAR(30) | NOT NULL, UNIQUE | 课程编号 |
| credit | DECIMAL(3,1) | DEFAULT 0.0 | 学分 |
| hours | INT | DEFAULT 0 | 课时 |
| description | VARCHAR(500) | — | 课程描述 |
| create_time | DATETIME | NOT NULL | 创建时间 |
| update_time | DATETIME | — | 更新时间 |

### 2.7 class_course — 排课表（班级-课程关联）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| class_id | BIGINT | NOT NULL, FK → class.id | 班级ID |
| course_id | BIGINT | NOT NULL, FK → course.id | 课程ID |
| teacher_id | BIGINT | NOT NULL, FK → sys_user.id | 授课教师ID |
| semester | VARCHAR(30) | NOT NULL | 学期（如 2024-2025-2） |
| create_time | DATETIME | NOT NULL | 创建时间 |

> 唯一约束：(class_id, course_id, semester)，同一班级同一学期不能重复安排同一门课。

### 2.8 score — 成绩表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| student_id | BIGINT | NOT NULL, FK → student.id | 学生ID |
| class_course_id | BIGINT | NOT NULL, FK → class_course.id | 排课ID |
| score | DECIMAL(5,2) | — | 分数（0.00~100.00） |
| exam_type | VARCHAR(20) | DEFAULT '期末' | 期中 / 期末 / 补考 |
| remark | VARCHAR(255) | — | 备注 |
| create_time | DATETIME | NOT NULL | 创建时间 |
| update_time | DATETIME | — | 更新时间 |

> 唯一约束：(student_id, class_course_id, exam_type)，同一学生同一课程同一考试类型不重复。

## 4. 核心查询场景

### 4.1 查询某班级所有学生
```sql
SELECT * FROM student WHERE class_id = ? AND status = 1;
```

### 4.2 查询某班级某课程的所有学生成绩
```sql
SELECT s.student_no, s.name, sc.score, sc.exam_type
FROM score sc
JOIN student s ON s.id = sc.student_id
WHERE sc.class_course_id = ?;
```

### 4.3 成绩统计（平均分/最高分/最低分/及格率）
```sql
SELECT
    COUNT(*) AS total,
    ROUND(AVG(sc.score), 1) AS avg_score,
    MAX(sc.score) AS max_score,
    MIN(sc.score) AS min_score,
    ROUND(SUM(CASE WHEN sc.score >= 60 THEN 1 ELSE 0 END) / COUNT(*) * 100, 1) AS pass_rate
FROM score sc
WHERE sc.class_course_id = ? AND sc.exam_type = '期末';
```

### 4.4 学生查看自己的成绩单
```sql
SELECT c.course_name, co.course_code, co.credit,
       cc.semester, sc.score, sc.exam_type
FROM score sc
JOIN class_course cc ON cc.id = sc.class_course_id
JOIN course co ON co.id = cc.course_id
JOIN class c ON c.id = cc.class_id
WHERE sc.student_id = ?;
```
